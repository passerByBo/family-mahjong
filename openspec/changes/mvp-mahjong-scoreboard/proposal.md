# Proposal: 家庭麻将计分板 MVP

## 概述

为家庭麻将游戏打造一个简洁易用的计分板应用。核心解决"打牌时忘记谁输谁赢"的痛点,支持快速记分、自动计算积分、庄家轮转和历史回看。

## 背景

- 家庭成员定期打麻将,使用推倒胡规则
- 目前靠记忆或纸笔记分,容易出错遗忘
- 需要一个手机上随时可用的计分工具

## 技术栈

- **前端**: Next.js 15 + Tailwind CSS + shadcn/ui
- **后端**: Next.js API Routes (Serverless)
- **数据库**: Cloudflare D1 (SQLite)
- **部署**: Cloudflare Pages + GitHub Actions
- **Monorepo**: pnpm + Turborepo

## 功能模块

### 1. 用户管理

- 创建玩家: 输入名称 + 选择预设头像
- 预设头像库: 12-16 个卡通/动物头像供选择
- 编辑/删除玩家
- 全局玩家列表

### 2. 牌局管理

- 创建牌局: 从玩家列表选 4 人
- 设置座位顺序
- 设置起始庄家
- 牌局列表: 进行中 / 已结束
- 继续未完成的牌局
- 手动结束牌局

### 3. 打牌界面 (核心)

#### 界面

- 麻将桌视图: 4 人围坐
- 每人旁边显示: 头像、名称、本轮积分
- 每人操作按钮: [杠] [胡] [自摸]
- 庄家标识
- 底部状态栏: 当前轮次、局数、庄家信息

#### 积分规则 (推倒胡)

| 事件 | 赢家 | 庄家 | 其他闲家 |
|------|------|------|----------|
| 杠 | +3 | -1 | 各-1 |
| 庄家胡 | +6 (庄) | — | 各-2 |
| 庄家自摸 | +12 (庄) | — | 各-4 |
| 闲家胡 | +4 (闲) | -2 | 各-1 |
| 闲家自摸 | +8 (闲) | -4 | 各-2 |

注: 杠是独立事件,一局内可多次杠,同一人也可多次杠。

#### 庄家规则

- 庄家赢了 → 连庄
- 庄家输了 → 下一位坐庄 (按座位顺序)
- 杠、流局不影响庄家

#### 轮次规则

- 一轮 = 从起始庄家开始,每人都坐过庄,且最后一个庄家输了
- 下一轮起始庄家 = 上一轮起始庄家的下一位 (按座位顺序)
- 轮结束时: 轮积分结算到牌局总积分,轮积分重置

#### 其他操作

- **撤销**: 可撤销上一次操作
- **中途换人**: 新人从 0 分开始;老人回来继承之前冻结的积分
- **换位次**: 随时可换,换后可重新设置起始庄家

### 4. 积分与统计 (MVP)

- 牌局总积分: 已结算轮的累计
- 本轮积分: 独立显示,未结算
- 展开查看每轮明细
- 历史牌局回看总分

## 数据模型

### Player (玩家)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT PK | 唯一标识 |
| name | TEXT | 玩家名称 |
| avatar | TEXT | 预设头像标识 |
| created_at | DATETIME | 创建时间 |

### Game (牌局)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT PK | 唯一标识 |
| status | TEXT | playing / finished |
| created_at | DATETIME | 创建时间 |
| finished_at | DATETIME | 结束时间 |

### GamePlayer (牌局玩家, 支持换人场景 >4人)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT PK | 唯一标识 |
| game_id | TEXT FK | 牌局 |
| player_id | TEXT FK | 玩家 |
| seat_position | INTEGER | 座位 1-4 |
| is_active | BOOLEAN | 是否在场 |
| joined_at | DATETIME | 加入时间 |
| left_at | DATETIME | 离开时间 (换人时) |

### Round (轮)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT PK | 唯一标识 |
| game_id | TEXT FK | 牌局 |
| number | INTEGER | 第几轮 |
| starter_id | TEXT FK | 起始庄家 |
| status | TEXT | playing / finished |
| created_at | DATETIME | 创建时间 |

### Hand (局)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT PK | 唯一标识 |
| round_id | TEXT FK | 所属轮 |
| dealer_id | TEXT FK | 当前庄家 |
| number | INTEGER | 本轮第几局 |
| created_at | DATETIME | 创建时间 |

### HandEvent (局内事件)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT PK | 唯一标识 |
| hand_id | TEXT FK | 所属局 |
| type | TEXT | kong / win / self_draw |
| player_id | TEXT FK | 操作玩家 |
| created_at | DATETIME | 创建时间 |

### ScoreChange (积分变化)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT PK | 唯一标识 |
| event_id | TEXT FK | 关联事件 |
| player_id | TEXT FK | 玩家 |
| amount | INTEGER | 积分变化 (+/-) |

## 非目标 (MVP 不做)

- 跨牌局统计 (按日期范围查询) → 二期
- 用户登录/注册体系
- 多设备实时同步
- 头像上传
- 麻将番数/牌型识别

## 页面结构

```
/                       首页: 进行中的牌局 + 新建牌局入口
/players                玩家管理列表
/players/new            创建玩家
/games/new              创建牌局 (选人、排座、设庄)
/games/[id]             打牌界面 (麻将桌视图)
/games/[id]/summary     牌局总结 (总积分 + 每轮明细)
/history                历史牌局列表
```

---

## 核心交互流程

```
创建玩家 → 创建牌局(选4人) → 设座位+起始庄家
  → 进入麻将桌 → 点击 [杠]/[胡]/[自摸]
  → 自动计算积分 → 自动切换庄家
  → 一轮结束 → 结算到总分 → 进入下一轮
  → 手动结束牌局 → 查看总结

中途可: 暂停(退出) / 换人 / 换位次 / 撤销
```
